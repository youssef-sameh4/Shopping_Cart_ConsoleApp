# Console Shopping Cart System

A **console-based shopping cart application** built in **C#** demonstrating **Object-Oriented Programming (OOP) principles**, interactive UI, and usage of collections like `Dictionary`, `List`, `Stack`, and `Tuple`.  

This project allows users to **add, view, remove items**, **checkout**, and even **undo last actions**, providing a simple yet functional shopping cart experience.

---

## Features

- Display a list of available products with prices.
- Add items to the cart by entering product names.
- View cart with product names and prices.
- Remove items from the cart.
- Checkout with total price calculation.
- Undo the last action (add/remove) using a stack.
- Fully interactive console UI with colored output for better readability.
- Implements **OOP concepts** such as classes, methods, and encapsulation.
- Uses C# **collections** effectively:  
  - `Dictionary` for product storage  
  - `List` for cart items  
  - `Stack` for undo actions  
  - `Tuple` for pairing items and prices
